
　　　　　　ナエエエ！！


■　動作環境
---------------------------------------------------------------
　Windows 10

■　起動
---------------------------------------------------------------
　「Naeee.exe」にて起動できます。

■　終了
---------------------------------------------------------------
　・Escapeキー(タイトル画面)
　・Alt+F4　の同時押し

■　操作方法
---------------------------------------------------------------
【キーボード】
　Spaceキー：ジャンプ、決定
  左右キー：回転
　Escキー : ポーズ

■　注意事項
---------------------------------------------------------------
　音量が大きくなっている場合がございます。初回起動の際は音量にご注意ください。
